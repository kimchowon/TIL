## Today I Learned; 오늘 내가 배운 것

## 🙆🏻‍♀️나의 TIL 방식🦜

- 그날 그날 공부한 것을 기록한다.
    - 눈으로 보기만 하는 것과 스스로 직접 코드도 짜보고, 내용을 정리하는 것은 천지 차이다.
    - 나의 공부 기록이므로 자유롭게 작성한다.
- TIL에 기록한 것 중 사람들과 공유하고 싶은 내용이 있다면 블로그에 포스팅한다.
    - 다른 사람이 볼 수 있는 글이므로 TIL에 기록한 내용에 더 추가하거나 삭제하는 등 다듬어 올린다.
